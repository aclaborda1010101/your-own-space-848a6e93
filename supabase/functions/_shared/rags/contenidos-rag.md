## COPYWRITING

### Fórmulas Clásicas

**AIDA (Atención, Interés, Deseo, Acción)**
```
Atención: Hook que detiene el scroll
Interés: Desarrollar el problema/oportunidad
Deseo: Mostrar la transformación
Acción: CTA claro y específico
```

**PAS (Problema, Agitación, Solución)**
```
Problema: Identificar el dolor
Agitación: Amplificar las consecuencias
Solución: Presentar la respuesta
```

**BAB (Before, After, Bridge)**
```
Before: Situación actual (problema)
After: Situación deseada (resultado)
Bridge: Cómo llegar de A a B
```

**4 Ps (Promise, Picture, Proof, Push)**
```
Promise: Promesa clara
Picture: Visualizar el resultado
Proof: Evidencia/testimonios
Push: Llamada a la acción
```

### Técnicas de Hooks

**Tipos de Hooks Efectivos**
1. **Pregunta provocadora**: "¿Y si todo lo que crees sobre productividad estuviera mal?"
2. **Declaración contraria**: "Deja de madrugar. No funciona para todos."
3. **Historia personal**: "Hace 3 años quebré mi empresa. Fue lo mejor que me pasó."
4. **Dato sorprendente**: "El 73% de emprendedores exitosos fallaron 3+ veces."
5. **Promesa específica**: "Cómo conseguí 1000 suscriptores sin gastar un euro."

**Elementos de un buen Hook**
- Específico (números, detalles concretos)
- Inesperado (rompe patrones)
- Personal (experiencia real)
- Útil (promete valor)
- Breve (menos de 10 palabras ideal)

### CTAs (Llamadas a la Acción)

**Principios**
- Un solo CTA por pieza
- Acción específica y clara
- Beneficio implícito o explícito
- Urgencia sin manipulación
- Reducir fricción

**Ejemplos por Objetivo**
```
Engagement: "¿Te ha pasado algo similar? Cuéntame en comentarios"
Guardados: "Guarda esto para cuando lo necesites"
Compartir: "Envíaselo a alguien que necesite leerlo hoy"
Seguir: "Sígueme para más reflexiones como esta"
Link: "Link en bio para la guía completa"
Suscripción: "Únete a los 5000 que reciben esto cada martes"
```

## STORYTELLING

### Estructura del Viaje del Héroe (Simplificada)

```
1. Mundo ordinario → El protagonista en su día a día
2. Llamada a la aventura → Algo cambia, surge un reto
3. Rechazo inicial → Dudas, miedos, resistencia
4. Encuentro con mentor → Ayuda externa (o insight interno)
5. Cruce del umbral → Decisión de actuar
6. Pruebas y aliados → El camino con obstáculos
7. Momento crítico → La prueba más difícil
8. Recompensa → Logro o aprendizaje
9. Regreso transformado → Vuelta con nuevo conocimiento
10. Compartir el elixir → Ayudar a otros con lo aprendido
```

### Storytelling para Redes Sociales

**Estructura Corta (Post)**
```
Hook → Contexto (2 líneas) → Conflicto → Giro → Lección → CTA
```

**Ejemplo**:
```
El lunes despidieron a mi jefe.

Llevaba 15 años en la empresa. Conocía cada proceso.
Era el más leal.

Y aun así, lo echaron.

Aprendí algo ese día:
La lealtad sin resultados no te protege.
Los resultados sin visibilidad tampoco.

Desde entonces, documento todo lo que hago.
No por ego. Por supervivencia.

¿Tú cómo haces visible tu trabajo?
```

### Elementos de Historia Efectiva

**Detalles Sensoriales**
- Describe lo que ves, oyes, sientes
- Momentos específicos > generalidades
- "A las 3AM, mirando el techo" > "una noche"

**Diálogo**
- Rompe el texto
- Humaniza la historia
- Crea ritmo

**Vulnerabilidad Calibrada**
- Comparte fracasos, no solo éxitos
- Muestra el proceso, no solo el resultado
- Admite dudas y miedos

**Tensión y Resolución**
- Crea conflicto (interno o externo)
- Mantén suspense moderado
- Ofrece cierre satisfactorio

## FRASES MOTIVACIONALES CERCANAS

### Estilo que Buscamos
- Reflexivas, no imperativas
- Preguntas que invitan a pensar
- Observaciones honestas sobre la vida
- Humor suave cuando encaja
- Reconocer la complejidad

### Ejemplos por Categoría

**Productividad sin presión**
- "No necesitas hacer más. Necesitas hacer menos, mejor."
- "A veces el mejor plan es no tener ninguno."
- "Descansar también es trabajar en ti."
- "Tu valor no se mide en tareas completadas."

**Desarrollo personal honesto**
- "Crecer duele. Y está bien admitirlo."
- "No tienes que tenerlo todo claro. Solo el siguiente paso."
- "El progreso no es lineal. Algunos días retrocedes para coger impulso."
- "Está bien no estar bien. Lo que no está bien es fingir que lo estás."

**Relaciones y conexión**
- "Las mejores conversaciones empiezan con '¿Cómo estás de verdad?'"
- "Menos consejos, más escuchar."
- "A veces lo que alguien necesita es que le digas: 'Yo también.'"

**Trabajo y carrera**
- "Tu trabajo no es tu identidad. Es algo que haces, no quien eres."
- "Decir que no a algo bueno te deja espacio para algo mejor."
- "El éxito de otros no te quita el tuyo."

**Paternidad/Maternidad**
- "No existen los padres perfectos. Solo los que lo intentan."
- "A veces la mejor crianza es perdonarte a ti mismo."
- "Lo que recordarán no es lo que les compraste, sino cómo les hiciste sentir."

## FORMATOS DE CONTENIDO

### Carrusel/Slides
```
Slide 1: Hook visual + título provocador
Slide 2-6: Contenido de valor (1 idea por slide)
Slide 7: Resumen/Takeaway
Slide 8: CTA + Firma
```

### Post Texto Largo
```
Hook (1-2 líneas)
[espacio]
Desarrollo (historia/contexto)
[espacio]
Punto de giro o insight
[espacio]
Lección aplicable
[espacio]
CTA conversacional
```

### Hilo/Thread
```
Tweet 1: Hook potente + promesa
Tweets 2-8: Un punto por tweet, numerados
Tweet 9: Resumen
Tweet 10: CTA + retweet del primero
```

### Newsletter
```
Asunto: Específico, curioso, beneficio claro
Intro: Personal, conectar con el lector
Cuerpo: 1-3 ideas desarrolladas
Cierre: Reflexión + CTA
P.S.: Bonus o recordatorio
```

## PROCESO CREATIVO

### Generación de Ideas
1. Experiencias personales recientes
2. Conversaciones que tuviste
3. Libros/podcasts que consumiste
4. Preguntas que te hacen frecuentemente
5. Errores que cometiste
6. Éxitos inesperados
7. Observaciones del día a día

### Estructura de Trabajo
```
1. Idea en bruto (capturar sin filtrar)
2. Pregunta clave: ¿Qué quiero que sienta/haga el lector?
3. Hook (dedicar tiempo aquí)
4. Estructura básica (puntos clave)
5. Primer borrador (sin editar)
6. Edición: cortar, simplificar
7. Revisión final: ¿suena a mí?
```

### Edición
- Elimina adverbios innecesarios
- Cambia voz pasiva por activa
- Acorta frases largas
- Lee en voz alta
- ¿Cada frase aporta valor?
- ¿El hook es lo suficientemente fuerte?

## ERRORES COMUNES

1. **Empezar con "Yo"**: Mejor empezar con el lector
2. **Demasiado contexto**: Ir al grano
3. **CTA genérico**: Ser específico
4. **Copiar sin adaptar**: Hacer tuyo el formato
5. **Inconsistencia**: Mejor poco frecuente y bueno
6. **Solo vender**: Dar valor real primero
7. **Ignorar comentarios**: El engagement es conversación
8. **Perfeccionismo**: Publicar es mejor que perfeccionar eternamente

## MÉTRICAS QUE IMPORTAN

### Vanity vs. Value
- Likes < Guardados/Compartidos
- Seguidores < Engagement rate
- Alcance < Conversaciones iniciadas
- Viralidad < Consistencia

### Señales de Contenido Bueno
- Comentarios con opiniones propias
- Gente etiquetando a otros
- Guardados y compartidos
- Mensajes directos de agradecimiento
- Conversaciones en los comentarios
